听到了傣历1372新年的通知。就我那点肤薄的知识，我只能说：傣历新年与我们过的新年是不一样的，泼水节之所以隆重，因为那是傣族人的新年。

我没弄明白1372是什么意思，又不想在网上像鬼子进村一样搜，如果自我拆度，1372也许是一个数学公式{% highlight python nowrap %} 3*7 {% endhighlight %}，刚好21。也许是告诉我们一个社会常识：不管三七二十一，还也许是本山哥的{% highlight c nowrap %} 1+3 {% endhighlight%} 在什么情况下等于7加2。

关于泼水节，笑话倒是许多。一哥们过完节后回来，在医院躺了一个月，朋友看他问怎么了，那哥们义愤填膺，怒气未消，冒着吐沫星子说：泼水就算了，还连药罐子都扔出来，药罐子扔出来也就算了，里面还带着熬得热的汤剂。
